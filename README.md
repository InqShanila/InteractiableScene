# Info
Unity version 6.3 LTS (6000.3.9f1)

Used assets:
- 3d Game Kit - https://assetstore.unity.com/packages/templates/tutorials/unity-learn-3d-game-kit-115747
- Animals FREE - Animated Low Poly 3D Models - https://assetstore.unity.com/packages/3d/characters/animals/animals-free-animated-low-poly-3d-models-260727
- AllSky Free - 10 Sky / Skybox Set - https://assetstore.unity.com/packages/2d/textures-materials/sky/allsky-free-10-sky-skybox-set-146014

### Controls
- WASD - movement
- Mouse XY - camera rotation
- Space - jump
- E - catch animal
- Left mouse button - attack
